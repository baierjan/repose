=======================================================================
                        Installing Repose/Repoq
=======================================================================


SUSE, openSUSE
==============

::

  zypper install cram curl haveopt make openssh sumaxy xmlstarlet zsh
  git clone https://github.com/openSUSE/repose.git
  mkdir repose-build
  cd repose-build
  ../repose/configure
  make check
  sudo make install
